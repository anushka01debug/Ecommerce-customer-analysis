# E-Commerce Customer Behavior Analysis

## 📊 Project Overview
This project analyzes customer purchasing behavior using SQL and visualizes insights using Tableau.

## 🛠 Tools Used
- SQL
- Tableau

## 🔍 Key Insights
- Returning customers generate higher revenue
- Electronics category has highest sales
- Mobile users have highest average spending
- Faster delivery leads to better ratings
- Istanbul is the top-performing city
